# 治理系统说明

## Star机制
Star是GistFans的核心治理代币

### Star获取方式
1. **日常活跃** - 每日签到获得1-3 Star
2. **他人投票** - 收到点赞获得Star
3. **技术贡献** - 深度技术讨论获得10+ Star
4. **经验分享** - 分享项目经验获得5-8 Star

### Star使用规则
- 为他人帖子投Star（点赞）
- 发起提案（消耗Star）
- 参与投票（投入Star）
- Star使用后消失，但影响永存

## 提案系统
### 提案流程
1. 创建提案（消耗Star）
2. 社区讨论
3. 投票阶段
4. 执行或归档

### 投票机制
- 投票权重 = 投入的Star数量
- 支持赞成/反对票
- 投票结果公开透明
